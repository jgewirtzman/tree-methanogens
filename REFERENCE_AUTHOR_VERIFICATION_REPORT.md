## Notes on methodology

- Author verification was performed by searching each DOI via web search and cross-referencing with publisher pages, PubMed, and institutional repositories.
- Direct CrossRef API access was unavailable due to network restrictions, so published author lists were obtained from web search results of publisher sites.
- For "et al." references, only the listed authors were compared against the corresponding positions in the published list.
- Minor accent differences (e.g., Glöckner, Bréchet, Mächler) were not flagged as mismatches.
- 4 references could not be verified (2 without DOIs, 1 bioRxiv preprint, 1 inconclusive lookup).
